Co to jest

Kompletne środowisko badawczo-rozrywkowe do Monopoly: grasz, symulujesz i trenujesz — wszystko lokalnie.

🧠

Silnik oceny na żywo

Setki dogrywek Monte Carlo z bieżącej pozycji (~30 ms) = realne szanse na wygraną. Pasek oceny i adnotacje ruchów jak w szachach.

🗺️

Analiza planszy

Heatmapa lądowań (łańcuch Markowa), ROI grup, najgorętsze pola. Matematyczne uzasadnienie strategii.

🧪

Laboratorium symulacji

Tysiące gier w przeglądarce: win-rate, bankructwa, histogramy, turnieje H2H z rankingiem Elo.

🏆

Wytrenowany model

Champion wypracowany ewolucyjnie (self-play, CEM) + polityka RL. Dominuje w stawce i w pojedynkach.

🔌

API i serwer REST

Podłącz własnego bota (Python lub dowolny język przez HTTP). Agenci MCTS i LLM (lokalne LM Studio).

Wytrenowany model „champion"

🏆

Trenowany self-play (CEM) na wielu panelach i liczbach graczy (anty-przeuczenie). Wypracował schemat: koleje + pomarańczowe + spółki + tanie sety, wysoka rezerwa gotówki, późne wychodzenie z więzienia.

Ogrywa każdą inną strategię w bezpośrednim pojedynku (kolejowy 80%, pomarańczowy 75%, monopolista 72%, agresor 88%). Ważne odkrycie: model nie transferuje między silnikami (przeuczenie) — przeglądarka ma osobnego, natywnie wytrenowanego championa.

Jak uruchomić

W przeglądarce — nic nie instalujesz: otwórz Monopoly Lab.

Symulator w Pythonie (bez zależności zewnętrznych):

git clone https://github.com/lkolo-prez/monopoly-lab.git
cd monopoly-lab
python3 run_sim.py 10000          # masowa symulacja + ranking strategii
python3 tournament.py             # turniej H2H, Elo, CSV
python3 train_evolve.py 30        # trenuj własny model (ewolucja self-play)
python3 server.py                 # serwer REST — podłącz własnego bota
python3 examples/llm_demo.py      # agent LLM przez lokalne LM Studio
